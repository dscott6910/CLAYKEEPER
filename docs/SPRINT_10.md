# Sprint 10 - Tournament Command Center

## Added

- Event-specific command center with event selector
- Event countdown and event date summary
- Check-in, squadding, and scoring progress meters
- Registration, enrollment, squad, and score KPI cards
- Squad operations board with Waiting, In Progress, and Complete states
- Per-squad score progress and completed-participant counts
- Readiness checklist with direct workflow links
- Upcoming shoot schedule
- Quick actions for Registration, Squadding, Live Scoring, and Leaderboard

## Database

No migration is required. This sprint uses the existing event, shoot, registration, squadding, and scoring tables.
