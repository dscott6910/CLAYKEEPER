# Sprint 8.0 - Performance and Reliability

## Changes

- Added route-level lazy loading for all major application modules.
- Added an accessible loading state while a module bundle is downloaded.
- Removed the mixed static/dynamic Supabase import in the Squadding module.
- Updated the frontend version to 0.7.0.

## Expected result

The initial JavaScript bundle is smaller and each feature is loaded only when opened. Navigation remains unchanged.
